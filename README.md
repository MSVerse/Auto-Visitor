# Auto-Visitor
Auto-Visitor adalah sebuah tools sederhana yang digunakan untuk meningkatkan jumlah pengunjung blog secara otomatis.

# Install
```
pkg install git
pkg install bc
git clone https://github.com/MSVerse/Auto-Visitor
cd Auto-Visitor
chmod 777 Auto.sh
./Auto.sh
```
